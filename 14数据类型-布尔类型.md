## 2.7 布尔类型bool
作用：布尔类型代表真或假的值

bool类型只有下面两个值：
+ true --- 真 (本质是1)
+ false --- 假（本质是0）

bool 类型变量 占 1个字节大小。

```cpp
bool flag = false;
cout << flag << std::endl;  // 0

flag = true;
cout << flag << std::endl;  // 1

// 本质上1代表真， 0代表假
std::cout << sizeof(bool) << std::endl; // 1
std::cout << sizeof(flag) << std::endl;  // 1
```
